                                          # 设置快捷键：

                                          先改成Eclipse的快捷键，然后修改

                                          快捷键 功能 路径

                                          Status Bar：状态栏 选择关闭 Alt + D

                                          Hide All Tool Windows：最大化当前窗口 Ctrl + M

                                          格式化代码，Reformat Code：Ctrl + Alt + Space

                                          编译代码，Make Project：Ctrl + B

                                          最后一次编辑的位置，Last Edit Location：Ctrl + R

                                          关闭文件，Close Tab：Ctrl + W

                                          Debug，Ctrl + F5，或者将热更新文件设置为此快捷键

                                          父类方法，接口方法，Go to Super Method，Ctrl + I

                                          Ctrl + J，当前文件中向上查找

                                          常用快捷键：

                                          # Ctrl

                                          Ctrl + A，全选

                                          Ctrl + B，编译代码

                                          Ctrl + C，复制

                                          Ctrl + D，删除一行

                                          Ctrl + E，查看历史文件，查看最近使用的文件，显示 IDEA 最近使用的一些文件。

                                          Ctrl + F，查看并替换内容

                                          Ctrl + G，查看实现，查看方法被使用的情况

                                          Ctrl + H，全局查找或者是根据后缀查找，关键字检索

                                          Ctrl + K，当前文件中向下查找

                                          Ctrl + L，跳转到文件中的具体一行

                                          Ctrl + M，已被设置成最大页面，Hide All Tool Window

                                          Ctrl + N，类似于右键点击

                                          Ctrl + O，查看文件方法

                                          Ctrl + P，打印

                                          Ctrl + Q，原先是最后一次编辑的位置，现在不设置

                                          Ctrl + R，没发现，设置成最近修改的文件，Recently Changed files

                                          Ctrl + S，保存

                                          Ctrl + T，Go to Implementation(s)查看实现类，查看方法/类的实现类，如果我们想直接跳转到某个方法/类的实现类，直接在方法名或者类名上使用快捷键，如果对应的方法/类只有一个实现类的话，会直接跳转到对应的实现类。

                                          如果对应的方法/类有多个实现类的话，IDEA 会弹出一个选择框让你选择。

                                          Ctrl + U，交换面板，debug情况下可以好看变量的值，设置为Quick Evaluate Expression（快速查看变量的快捷键）

                                          Ctrl + Shift + Enter，添加到Watches中

                                          Ctrl + Shift + I，在Watches中把某个变量在弹框中观察

                                          Ctrl + V，粘贴

                                          Ctrl + W，原先是全屏，现在转为关闭文件

                                          Ctrl + X，剪切

                                          Ctrl + Y，重复上一动作

                                          Ctrl + Z，回到上一步

                                          Ctrl + /，单行注释

                                          // 注释

                                          Ctrl + F10，Run context configuration，运行上线文配置，用于运行测试类，运行main函数

                                          Ctrl + Enter，当前行后面添加一行

                                          # Alt

                                          Alt + /，代码提示

                                          Alt + 7，展示代码结构、类结构：类结构可以让我们快速了解到当前类的方法、变量/常量，非常使用！

                                          Alt + Enter，导入类、包，包括提供建议：例如equals翻转Flip'.equals()'

                                          Alt + 上下，上下移动代码，可以是一行，也可以是多行

                                          Alt + Insert，生成各种方法

                                          # Shift

                                          Shift + Enter，当前行前面添加一行

                                          Shift + Shift，无处藏身，全局查找数据

                                          Shift + F1，Previous Highlighted Error: 上一个错误

                                          Shift + F2，Next Highlighted Error: 下一个错误

                                          # Ctrl + Shift

                                          Ctrl + Shift + R，全局查找文件，可以快速检索类/文件。Go to File

                                          Ctrl + Shift + U，大小写变化

                                          Ctrl + Shift + H，全局替换

                                          Ctrl + Shift + O，优化包引入

                                          Ctrl + Shift + W：关闭所有文件

                                          Ctrl + Shift + C， Clear All，清空所有日志

                                          Ctrl + Shift + F10，Debug context configuration，Debug上线文配置，用于Debug测试类，Debug main函数

                                          Ctrl + Shift + /，多行注释，也可以取消多行注释，先输入注释然后快捷键

                                           /* 注释1 。。。 注释n */

                                          # Alt + Shift

                                          Alt + Shift + D，Rebel Debug

                                          Alt + Shift + C，打开接口调用，来源于EasyApi插件，可以复制接口和传参key用于各种测试

                                          # Ctrl + Alt

                                          Ctrl + Alt + 下，复制一行代码

                                          Ctrl + Alt + /，快速搜索Controller的接口，来源于RestfullTool插件

                                          # GIT

                                          Alt + Shift + F：git fetch

                                          Ctrl + Shift + P：git pull

                                          Ctrl + 向上箭头，Privious Difference，上一个不同

                                          Ctrl + 向下箭头，Next Difference，下一个不同

                                          Ctrl + Alt + K，git commit && git push

                                          Ctrl + Shift + K，git push

                                          Ctrl + Alt + Shift + J，Aliababa编码规范扫描，需要去除原有的快捷键占用

                                          # Other

                                          F2，重命名

                                          psvm为main方法的快捷键

                                          public static void main(String[] args) {

}

                                          sout为控制台输出的快捷键

                                          System.out.println();

                                          # F11系列

                                          模拟浏览器快捷键习惯

                                          1. 敲代码和关注消息：Enter(Exist) Distraction free Mode(进入或退出无干扰模式设置为Ctrl + Shift + F11)，不显示代码列和其他工具栏
                                          2. Enter(Exist) Presentation Mode(进入或退出演示模式，世界里只有代码和代码行，设置为Ctrl + F11)，适合敲代码，可以设置字体大小
                                          3. Enter Full Screen：全屏 F11
                                          4. 忘我模式敲代码：Enter(Exist) Presentation Mode + Enter(Exist) Distraction free Mode，1+2结合进入敲代码忘我模式，世界里只有代码，字体大小默认28pt，可以放大缩小
                                          5. 忘我模式小字体敲代码：Enter Full Screen + Enter(Exist) Distraction free Mode，1+3结合进入敲代码忘我模式，世界里只有代码，字体大小默认自己设置的pt，可以放大缩小，推荐
